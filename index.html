pylon crystal

Pylon Form

Pylon has 2 states, Static (Static) and Dynamic (Morph):

  • In Static state, Pylon will take on the image of an energy crystal, with the color depending on the support feature.
  • Pylon can switch into a “Morph” state, which means it transforms into another game-compatible object (eg dragons or spaceships etc).

Each pylon can only morph into 1 object in different games. This object color is also specified by the crystal color.

Pylon Effect

Players who equip Pylon will recevie buffs depend on the type of Pylon and the game being played

Static: Using Pylon’s “Static” state is like using a crystal to insert gems into a weapon.

Morph: Use Pylon’s “Morph” state to transform into an enquivalent in-game item i.e spaceship in 3D spacehooter game or create a “fire bullet” effect for a spaceship.

Pylons can combine energies to create a new pylon

Multiserve

Once the GateGame ecosystem is activated, with more Pylons put into the network and the larger
the energy generated, when reaching the overload threshold, the space dimension will be broken
and create portals to other dimensions.

Mutiserve Portal Key

To enter the Multiverse: the Portal key is required – which can be obtained from the Minivault or GG Box, special GateGame events, or trade.

On the Portal Key, there will be sockets for mounting different “Source Crystal”. These “Source Crystal” will spawn when the player plays games in the GG Metaverse (while equipping Pylon) according to the criteria of time, achievement and randomness.

Each Multiverse Transverse Portal will require a Portal Key with a different number of “Source Crystal”, the higher the Portal level, the more “Source Crystal” will be required (even more Portal Keys)

Beyond the Portal: This dimension can be parallel worlds (parallel worlds) or a completely different world. This will give rise to the Multiverse system with the properties “Unique”, “Unpredictable”. The player can pass through this Portal will have access to more complex worlds and bring more advantages/ generate more resources/ token/ items.

Pylon’s Level & Minivault

When open the vault the user will receive the Pylons randomly. The higher the minivault level, the higher the level Pylons will be granted.

The higher the level Minivault is, the more likely the Minivault will have Portal Key/Source Crystal.
Each Minivault will have a special pylon with a unique color and visual effect that will never appear again.

User can combine Minivault to become 1 Vault.
